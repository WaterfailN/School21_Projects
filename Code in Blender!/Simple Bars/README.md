# Long story short
It turns out that you can also do visualizations in a blender! Of course, there are already addons for this, created by the wonderful community. However, it was interesting for me to try to explore this topic on my own

# Вкратце
Оказывается, в блендере можно тоже делает визуализации! Конечно, для этого уже есть аддоны, созданные замечательным сообществом. Однако мне было интересно попробовать самостоятельно изучить этот вопрос

# How to use
1. Download blender from the official site ([blender.org](https://www.blender.org/)) or from steam
2. In the .blend file change the "filepath" to the path of .csv file
3. Delete all the object (Press A and Del in the 3d window)
4. Start the script (play button in the upper right corner of the text editor window)
5. Watch the magic!
